Part 3

Basic raster creation and calculation

Section a:

Create a first raster (30 rows x 30 columns) and assign values to the raster based on random values from an uniform distribution. Plot the raster and histogram.

library(raster)

ras <- raster(nrows=30,ncols=30,xmn=0,xmx=30,ymn=0,ymx=30)
ras[] <- runif(ncell(ras))
plot(ras)

hist(ras)

Section b:

Create a second raster (30 rows X 30 columns) and assign values to the raster based on random values from a normal distribution. Plot the raster and histogram.

ras2 <- raster(nrows=30, ncols=30, xmn=0, xmx=30, ymn=0, ymx=30)
ras2[] <- rnorm(ncell(ras2))
plot(ras2)

hist(ras2)

Section c:

Create a third raster (30 rows X 30 columns) and assign values to the raster based on cell-by-cell addition of the first raster and the second raster. Plot the raster and histogram.

ras3 <- ras+ras2
plot(ras3)

hist(ras3)

Section d:

Calculate the mean value of the third raster. Reclassify the third raster into a binary image: 1(cell value>mean value) and 0 (cell value<=mean value). Saved the reclassified image to your working directory as a TIFF image.

cellStats(ras3,"mean")
